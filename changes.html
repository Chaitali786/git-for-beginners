Git Changes Guide

Update a repo

git pull→ get new changes

git pull origin branch-name→ pull specific branch

Review Changes

git status→ see changed files

git diff→ see exact changes

git log→ see commit history

Add Changes

git add file-name→ add one file

git add.→ add all files

git commit -m "message"→ save with a message

Push Changes

git push→ send your changes to GitHub

git pull origin branch-name - pull specific branch

Remove Changes

git reset--soft Head~1→ undo last commit(keep files)

git reset --hard Head~1 - delete commit & changes
